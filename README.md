Excel file Heavymetal_sources_git.xlsx contains data of source profile.
Excel file IGP_test_bihar_git.xlsx contains data of fog samples. Sample ID "A01" to "A11" is for location Arthauli and sample ID "M01" to "M10" is for location Muzaffarpur.
MLP_fc.ipynb file computes source apportionment using Multi Layer Perceptron model.
QR_fc.ipynb file calcultes source apportionment using QR decomposition method.
RF_fc.ipynb file computes source apportionment using Random Forest Regressor model.
